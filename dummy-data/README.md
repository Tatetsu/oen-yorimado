# dummy-data

監査対応用ダミーデータ生成ディレクトリ。

## ディレクトリ構成

```
dummy-data/
├── 児童別/
│   ├── 実績報告書/                    # 書類1: 児童ごとの実績報告書
│   │   ├── 石神直都_9200836808_2025-12.csv
│   │   ├── 富永日向_9200757988_2025-12.csv
│   │   └── ...
│   └── 重度支援加算/                  # 書類2: 重度対象児童のみ
│       ├── 溝口一花_9200364314_2026-02.csv
│       ├── 林夏渚_9200809896_2026-02.csv  ← 睡眠22時特例
│       └── ...
├── 実績報告書_全児童.csv              # 書類1マージ結果
├── 重度支援加算_全児童.csv            # 書類2マージ結果
├── generate_from_pdf.py
└── README.md
```

## 再申請時の運用

児童別CSVは `児童名_受給者証番号_YYYY-MM.csv` の命名規則。
再申請で同じ児童・同じ月のデータを再生成すると個別CSVが上書きされる。
マージ時は (受給者証番号, 日付) をキーに、ファイル更新日時が新しい方を採用。

## 書類1: 実績報告書

```
日付,受給者証番号,児童名,体温,夕食,朝食,昼食,入浴,便,服薬(夜),服薬(朝),その他連絡事項
```

| 項目 | 値 | 備考 |
|---|---|---|
| 日付 | YYYY-MM-DD | 令和→西暦変換済み |
| 体温 | 36.0〜37.5 | 正規分布、平均36.5 |
| 食事（夕食・朝食・昼食） | 完食/半分/食べなかった/− | 重み付きランダム |
| 昼食（他サービス併給時） | − | 別施設で昼食のため |
| 入浴 | ○/× | 90%で○ |
| 便 | ○/× | 70%で○ |
| 服薬（夜・朝） | ○/× | 85%で○ |
| その他連絡事項 | 下記リストからランダム | |

## 書類2: 重度支援加算チェック記録

```
日付,受給者証番号,児童名,12:30,17:00,18:00,...,8:00,睡眠(21:00),睡眠(4:00),睡眠(7:00)
```

| 項目 | 値 | 備考 |
|---|---|---|
| 12:30 | ○/− | 他サービス併給→「−」、それ以外→「○」 |
| 17:00〜翌8:00（毎時16列） | ○/× | 98%で○ |
| 睡眠(21:00) | ○/× | 95%で○。林夏渚のみ22:00 |
| 睡眠(4:00) | ○/× | 75%で○（たまに×） |
| 睡眠(7:00) | ○/× | 95%で○ |
