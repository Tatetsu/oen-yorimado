# スプレッドシート利用手順書

Yorimadoの来館記録を管理するスプレッドシートの使い方をまとめた手順書です。
パソコンの操作に慣れていない方でも迷わないように、画面の画像付きで説明しています。

---

## この手順書の読み方

- **毎日・毎月やること**（01〜05）から順に読むと、普段の作業の流れが分かります。
- **登録や設定を変えるとき**（10〜13）は、必要になったときだけ開いてください。
- 画像は実際の画面と同じものです。赤い枠で囲まれている場所が「クリックする場所」です。

## 画面の全体像

スプレッドシートを開くと、下のようにたくさんのタブが並んでいます。
それぞれのタブが「シート」と呼ばれる1つのページです。

| タブの名前 | 何のためのシート？ |
|---|---|
| 未記入児童一覧 | 来館予定なのにフォーム送信されていない児童を確認 |
| フォームの回答 | スタッフが送信したフォーム回答が集まる場所 |
| →マスタ | マスタ関連の見出し（クリック不要） |
| 設定 | メール文面や営業日などの基本設定 |
| 許可ユーザー | スタッフ用リンク（トークン）を発行する場所 |
| 児童マスタ | 児童の基本情報を登録する場所 |
| スタッフマスタ | スタッフの基本情報と稼働日を登録する場所 |
| 定型文マスタ | 連絡事項でよく使う文言を登録する場所 |
| →集計結果 | 集計関連の見出し（クリック不要） |
| 月別集計 | 月ごとの利用状況をまとめたシート |
| 確定来館記録 | 実際の来館記録と振り分け予測をまとめたシート |
| 来館カレンダー | カレンダー形式で来館状況を見るシート |
| 児童別ビュー | 特定の児童の記録だけを絞り込んで見るシート |
| ログ | エラー記録（普段は見なくてOK） |

---

## 困ったとき

- スプレッドシートを開き直す → タブを一度閉じて、もう一度開くだけで直ることが多いです。
- 数字が出ない・空白になる → 対象年月のプルダウンが「すべて」になっていないか確認してください。
- それでも直らないとき → 管理者に連絡してください。

---

## 大事なお願い

以下のシート・セルは**触らないでください**。数式が壊れると、全体の計算がおかしくなります。

- **設定シート**の値（メール文面を変えたいときだけ → [13_設定を変更する.md](13_設定を変更する.md) 参照）
- **確定来館記録**の「データ区分」列（「実データ」「振り分け」を区別する大事な印です）
- **月別集計・来館カレンダー**の表の中の数字（自動で計算されます）
- **ログシート**（エラー記録が自動で入る場所）
