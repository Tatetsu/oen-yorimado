# 00. フォームを入力する

## このページでやること

スタッフがGoogleフォームから来館記録を送信するときの**入力ルール**をまとめています。
1回の来館（単泊・複数日宿泊いずれも）について **フォーム1行で入所〜退所までを記録** します。

- **いつやるか**：来館記録を送信するとき（基本は退所した翌朝）
- **かかる時間**：1件あたり1〜2分
- **誰がやるか**：その来館の担当スタッフ

---

## 基本ルール（すべての回答に共通）

| 項目 | ルール |
|---|---|
| スタッフ1 | 必須。担当者本人を選ぶ |
| スタッフ2 | 任意。複数担当のときだけ選ぶ |
| 児童名 | プルダウンから選ぶ（手入力禁止） |
| 体温・食事・服薬・入浴・便 | 該当する時間帯のものだけ入力。なければ「-」 |
| 入眠時刻・朝4時チェック・起床時刻 | **必須**。プルダウンから選択（入眠時刻=20:30〜22:00 / 10分刻み、朝4時チェック=睡眠 or 覚醒確認後に付き添い、起床時刻=6:00〜7:30 / 10分刻み） |
| その他連絡事項 | 自由記入。引き継ぎや特記事項を記載 |

> **「記録日」項目は廃止されました（2026-04 後半）**：フォーム送信時の**送信日（タイムスタンプの日付）**が「利用日」として扱われます。**退所した翌朝に送信** すれば前夜・前日分の記録として正しく扱われ、メール通知も毎朝8時にその朝の送信分が拾われます。
>
> **送信タイミング**：原則「退所した翌朝」。1回の来館（単泊・複数日宿泊いずれも）につき**1件**を送信してください（毎朝のメール通知は「過去24時間以内に送信された未送信のフォーム回答」を対象に発行されます）。

---

## 入所日時・退所予定日時の入力ルール

**入所日時・退所予定日時はどちらも必須**です。1回の来館の入所〜退所をフォーム1行で記録します。

| 項目 | 値 |
|---|---|
| 入所日時 | **入力する**（来館した日時） |
| 退所予定日時 | **入力する**（帰った／帰る予定の日時） |

### 例

| 来館形態 | 入所日時 | 退所予定日時 |
|---|---|---|
| 単泊（同日帰宅） | 4/1 17:00 | 4/1 21:00 |
| 1泊2日 | 4/1 17:00 | 4/2 08:00 |
| 2泊3日 | 4/1 17:00 | 4/3 08:00 |
| 3泊4日 | 4/1 17:00 | 4/4 08:00 |

> **ポイント**：複数日にまたがる宿泊でも、フォーム送信は **退所した日の翌朝に1件だけ** です。途中の日に毎朝送る必要はありません。月別集計・利用枠カウントは入所日〜退所予定日の日付範囲を自動で展開して各日付をカウントします。

---

## 食事・服薬の入力ルール（2026-04 仕様）

行政様式（実績記録票）に合わせて、食事と服薬は**時間帯ごとに分かれて**います。

| 項目 | 入れるタイミング |
|---|---|
| 夕食 | 夕食を提供したとき |
| 朝食 | 朝食を提供したとき |
| 昼食 | 昼食を提供したとき（**他サービス併給時は「-」**） |
| 服薬(夜) | 夜の服薬があったとき |
| 服薬(朝) | 朝の服薬があったとき |

> **該当時間帯がない場合**：その項目は「-」を選んでください。空欄でも問題ありませんが、「-」のほうが「該当なし」が明確です。

---

## 睡眠まわりの入力ルール（2026-04 仕様）

旧「睡眠」(○/×) は廃止し、**3項目すべて必須プルダウン**で入力します。

| 項目 | 選択肢 | 入れるタイミング |
|---|---|---|
| 入眠時刻 | 20:30 / 20:40 / 20:50 / 21:00 / 21:10 / 21:20 / 21:30 / 21:40 / 21:50 / 22:00 | 児童が入眠した時刻に最も近い10分刻みを選ぶ |
| 朝4時チェック | 睡眠 / 覚醒確認後に付き添い | 4時の見回り時の状態を選ぶ |
| 起床時刻 | 6:00 / 6:10 / 6:20 / 6:30 / 6:40 / 6:50 / 7:00 / 7:10 / 7:20 / 7:30 | 児童が起きた時刻に最も近い10分刻みを選ぶ |

> **特例（林夏渚さん）**：入眠時刻は **`22:00`** を選んでください（コード上の特例扱いは廃止し、運用ルールで揃えます）。

## よくあるトラブル

| 症状 | 原因と対処 |
|---|---|
| 複数日宿泊なのに月別集計で1日しかカウントされない | 入所日時・退所予定日時のどちらかが正しく入っていない可能性（同じ日付になっている等）。Webビュー（修正ツール）で修正してください |
| 児童名のプルダウンに名前がない | 児童マスタに未登録、または「停止」状態の可能性。管理者に [10_児童を追加する.md](10_児童を追加する.md) で確認を依頼 |
| スタッフ名のプルダウンに自分の名前がない | スタッフマスタに未登録の可能性。管理者に [11_スタッフを追加する.md](11_スタッフを追加する.md) で追加を依頼 |
| 間違えて送信してしまった | 直接スプレッドシートを書き換えず、Webビュー（修正ツール）で修正・削除してください |

---

## 大事なお願い

- **送信は退所した翌朝に行ってください**。フォーム送信日が「利用日」として扱われます（記録日項目は廃止されました）。
- 入所日時・退所予定日時を正しく入れないと、月別集計・利用枠カウントがズレます。
- 送信した内容を直す場合は、**スプレッドシートを直接編集しない**でください（Webビューで修正）。
