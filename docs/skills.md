# 使用技術スタック

## プラットフォーム

| 技術 | 用途 |
|---|---|
| Google Sheets | データストア・閲覧UI・印刷出力 |
| Google Forms | スタッフの来館記録入力インターフェース |
| Google Apps Script | ビジネスロジック全般（集計・振り分け・データ統合） |
| Gmail API (via GAS) | 保護者への来館報告メール送信（Phase 3） |

## 開発言語

| 言語 | バージョン | 用途 |
|---|---|---|
| JavaScript (ES6+) | V8ランタイム | GAS全般 |

## GAS API・サービス

| サービス | 用途 | 備考 |
|---|---|---|
| SpreadsheetApp | シートの読み書き・書式設定 | メイン使用 |
| FormApp | フォームの選択肢同期 | 実装済み |
| ScriptApp | トリガー管理（時間・フォーム送信） | |
| GmailApp | メール送信 | Phase 3 |
| Utilities | 日付操作・フォーマット | |
| LockService | 同時実行制御 | 必要に応じて |

## 制約・制限（GAS無料枠）

| 項目 | 制限値 | 備考 |
|---|---|---|
| スクリプト実行時間 | 6分/実行 | 規模的に問題なし |
| トリガー実行回数 | 90分/日 | 問題なし |
| メール送信数 | 100通/日 | Phase 3で使用、規模的に問題なし |
| スプレッドシート容量 | 1000万セル | 問題なし |

## 開発ツール

| ツール | 用途 |
|---|---|
| Apps Script エディタ | GASコード編集・実行・デバッグ |
| Claude Code | 設計書作成・GASコード生成 |
| Git (ローカル) | 設計書・コードのバージョン管理 |
