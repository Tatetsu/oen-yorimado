## 仕様メモ（2026-03-16 ヒアリング）

### 年間利用枠と振り分けロジック
- 児童1人につき年間180回の利用枠がある
- 180÷12 = 約15回/月 → **月16回をMAX**として振り分ける
- 基本は月16回MAXで入れ、年度末に不足分を追加調整するイメージ
- 若干の変動は手動調整の想定

### シートの年単位切り替え
- シートは年単位で新しく切り替える運用
- 現在の年プルダウン（「全て」「2026年」等）は不要になる
- 過去年に遡って作成することはない

### 開館曜日（営業日）
- 開館している曜日は固定で決まっている
- 振り分けは開館曜日（営業日）のみに行う
- 手動での来館曜日選択時も開館曜日から選ぶ形にする

### スタッフ配置ルール
- 1日の来館枠がMAX（7人）の場合、スタッフ2人が必須
- Googleフォームに「担当スタッフ1」「担当スタッフ2」の枠を追加する
- 7人埋まっている場合にスタッフ1人しか選ばれなかったら、固定メンバーを「担当スタッフ2」に自動設定する
