# 集計ロジック 仕様書

## 概要

月別集計・児童別ビュー・来館カレンダーのデータ取得・表示範囲は「スコープ」によって統一的に制御される。

---

## スコープの仕組み（`buildScope_`）

各ビューシートの B1（年）・B2（月）の組み合わせでスコープを決定する。

| B1（年） | B2（月） | スコープ種別 | 対象範囲 |
|---------|---------|------------|---------|
| 具体値 | 具体値 | `month` | 指定年月のデータ |
| 具体値 | すべて | `year` | 指定年の全月データ |
| すべて | 具体値 | `month_all_years` | 全期間の指定月データ |
| すべて | すべて | `all` | 全期間データ |

実装: `utils.gs` → `buildScope_`, `matchesScope_`

---

## 月別集計（F-02）

実装: `gas/monthly-summary.gs`

### データソース

**フォームの回答（実記録のみ）**。振り分けデータは含まない。

### 利用枠の切り替え

| スコープ種別 | 使用する枠 | 参照列 |
|------------|----------|-------|
| `month` / `month_all_years` | 月間利用枠 | 児童マスタ J列 |
| `year` / `all` | 年間利用枠 | 児童マスタ I列 |

### カウント方法

`countVisitsByScope_` → `expandStayToDates_` で連泊を日ごとに展開し、展開後の各日付が `matchesScope_` を通過した分のみカウントする。

```
例: 2泊3日（3/1〜3/3）で3月スコープ
  → [3/1, 3/2, 3/3] が matchesScope_ を通過 → 3カウント
```

### 出力列

| 列 | 内容 |
|----|------|
| A | No.（児童マスタ） |
| B | 児童名 |
| C | 利用枠（スコープによって月間/年間を切替） |
| D | 来館数 |
| E | 残数（C - D） |
| F | 利用率（D / C、%表示） |

### 更新タイミング

- B1 または B2 セル変更時（onEdit トリガー）
- 手動ボタン押下時
- 振り分け実行後（`allocateRemainingPoints_` の末尾から `updateMonthlySummary()` 呼び出し）

## 確定来館記録（F-03）

実装: `gas/confirmed-visits.gs`

### データソース

**確定来館記録シート**（実データ + 振り分けデータの統合）

### 表示フィルタ

B1/B2 の年月に対応する行を `filterConfirmedVisits_` で表示/非表示切り替え（行ごとの hide/show）。

### デフォルト表示

シート初期セットアップ時に B1=前月の年, B2=前月月 がセットされる。

## 児童別ビュー（F-04）

実装: `gas/child-view.gs`

### データソース

**確定来館記録シート**（実データ + 振り分け）

### ドロップダウン構成

| セル | 選択肢 | 備考 |
|-----|-------|------|
| B1 | 全児童名（稼働/休止が先頭、退所済みが後半） | `getAllChildNameOptions()` |
| B2 | 年（すべて / YYYY年 降順） | `generateYearOptions()` |
| B3 | 月（すべて / 1月〜12月） | `generateMonthOptions()` |

### 基本情報エリア（4〜8行目）

| 項目 | データソース |
|------|----------|
| 保護者名 | 児童マスタ |
| 担当スタッフ | 児童マスタ |
| 月間利用枠 | 児童マスタ |
| 医療型の有無 | 児童マスタ |
| 来館回数 / 残枠 / 利用率 | 確定来館記録から集計 |

### 更新タイミング

- B1 / B2 / B3 セル変更時（onEdit トリガー）
- 手動ボタン押下時

## 来館カレンダー

実装: `gas/visit-calendar.gs`

### データソース

**確定来館記録シート**（実データ + 振り分け）

### レイアウト

```
1行目: タイトル（YYYY年 来館カレンダー 等）
2行目: (空白)
3行目: ヘッダー（日付 | 曜日 | 児童1 | 児童2 | ... | 日計）
4行目〜: 日別データ（○=実データ、△=振り分け）
```

### スコープ

B1（年）のみ参照（年単位表示）。B2は存在しない。

### 更新タイミング

- B1 セル変更時（onEdit トリガー）
- 月次一括処理後（`runMonthlyProcessAutomatic` / `runMonthlyProcess`）

---

## ドロップダウン年選択肢の生成

`generateYearOptions()` はフォームの回答シートの記録日列から年を収集し、降順で返す。

```
["すべて", "2026年", "2025年", ...]
```

データがない場合は現在の年1件のみ返す。
