# フォーム回答の修正ツール 操作マニュアル

## 概要

Googleフォームで送信された来館記録の回答を、ブラウザ上で修正・削除できるツールです。
誤送信や入力ミスが発生した場合に使用します。

---

## アクセス方法

管理者から共有されたGASウェブアプリのURLをブラウザで開いてください。
スマートフォン・PCどちらからでも操作できます。

---

## 画面構成

### タブ

画面上部に2つのタブがあります。

| タブ | 説明 |
|------|------|
| 年度ビュー | 指定した年度のデータを一覧表示 |
| 月別ビュー | 指定した年・月のデータを一覧表示 |

### フィルターバー

タブの下に絞り込み条件と「表示する」ボタンがあります。
条件を選択して「表示する」をタップするとデータが読み込まれます。

### 記録カード

読み込まれた回答が1件ずつカード形式で表示されます。  
各カードには以下の情報が表示されます。

- 利用日（フォーム送信日 = タイムスタンプの日付）・児童名
- スタッフ1・スタッフ2
- 入所・退所時刻
- 体温・食事・入浴・入眠時刻・朝4時チェック・起床時刻・便・服薬
- 連絡事項
- フォーム送信日時

---

## 操作手順

### 記録を表示する

1. タブ（年度ビュー または 月別ビュー）を選択する
2. 年度または年・月をプルダウンで選択する
3. 「表示する」をタップする
4. 一覧が表示される（右上に件数バッジが出ます）

### 記録を修正する

1. 対象のカードの「修正」ボタンをタップする
2. 修正フォーム（モーダル）が開く
3. 変更したい項目を編集する
4. 「修正を保存する」をタップする
5. 「修正しました。」と表示されれば完了

**修正できる項目**

| 項目 | 種別 |
|------|------|
| 児童名 | 必須（プルダウン選択） |
| スタッフ1・スタッフ2 | 任意 |
| 入所日時・退所日時 | 任意 |
| 体温 | 任意 |
| 食事・入浴・便・服薬 | プルダウン選択 |
| 入眠時刻・朝4時チェック・起床時刻 | プルダウン選択（必須）。入眠時刻=20:30〜22:00 / 10分刻み、朝4時チェック=睡眠 or 覚醒確認後に付き添い、起床時刻=6:00〜7:30 / 10分刻み |
| 連絡事項 | 任意 |

> **「記録日」入力欄は廃止（2026-04 後半）**：フォームから記録日項目を廃止し、タイムスタンプの送信日を利用日として扱う運用に変更したため、編集モーダルからも記録日入力欄を削除しています。保存時はスタッフ1（3列目）以降の17列のみが書き換えられ、既存の RECORD_DATE 列値は保護されます（旧データが入っていてもそのまま残ります）。

### 記録を削除する

1. 対象のカードの「削除」ボタンをタップする
2. 確認ダイアログが表示される
3. 内容を確認して「削除する」をタップする
4. 「削除しました。」と表示されれば完了

> **注意:** 削除した記録は元に戻せません。

---

## エラーが出たとき

| メッセージ | 対処方法 |
|-----------|---------|
| 読み込みエラー | ページを再読み込みして再試行してください |
| 保存に失敗しました | 時間をおいて再度「修正を保存する」をタップしてください |
| 削除に失敗しました | 時間をおいて再度お試しください |
| 初期化エラー | ページを再読み込みしてください。解消しない場合は管理者に連絡してください |
