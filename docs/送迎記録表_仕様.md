# 送迎記録表 仕様書

## 概要

児童施設「よりまど」の送迎記録表を、実績報告書CSV（児童別・日別）から自動生成する。

## 生成スクリプト

`scripts/generate_sougei_record.py`

### 実行方法

```bash
python3 scripts/generate_sougei_record.py
```

### 入力

- `dummy-data/児童別/実績報告書/` 配下のCSVファイル
- ファイル名規則: `{児童名}_{受給者証番号}_{YYYY-MM}.csv`
- エンコーディング: UTF-8 BOM付き

### 出力

- `dummy-data/送迎記録表_{YYYY-MM}.csv`

---

## CSV列定義

| 列名 | 内容 | 値 |
|---|---|---|
| 日付 | 実績報告書の日付と一致させる | YYYY-MM-DD |
| 曜日 | 日付に対応する曜日 | 月〜日 |
| 送迎区分 | 送迎の種類 | 夕迎え / 朝送り / 昼送り |
| 順番 | その送迎区分内での児童の順番 | 1〜N |
| 児童名 | フルネーム | 実績CSVの児童名 |
| 送迎職員 | 運転担当 | 只津（固定） |
| 介助者 | 同乗の介助者 | 溝口（固定） |
| 送迎方法 | 移動手段 | 車（固定） |
| 車番 | 車両番号 | 707（固定） |
| 点呼確認による異常 | 疾病・疲労・飲酒等 | なし（固定） |
| 開始時刻 | 送迎開始時刻 | H:MM |
| 終了時刻 | 送迎終了時刻 | H:MM |
| 備考 | 特記事項 | 通常は空欄 |

---

## 送迎パターンと時間設定

### 夕迎え（夕方のお迎え）

全出席児童が対象。施設まで車で迎えに行く。

| 項目 | 値 |
|---|---|
| 開始時刻 | 17:30〜17:40（ランダム） |
| 終了時刻（通常） | 18:20 / 18:30 / 18:40（ランダム） |
| 終了時刻（寺田/園田あり） | 18:40（固定） |

### 朝送り（朝の送り出し）

朝食ありの児童が対象（併給で朝から利用している児童）。

| 項目 | 値 |
|---|---|
| 開始時刻 | 7:40〜7:50（ランダム） |
| 終了時刻 | 8:40〜8:45（ランダム） |

### 昼送り（昼食後の送り出し）

昼食ありの児童が対象（併給なしで昼食後に帰宅する児童）。

| 項目 | 値 |
|---|---|
| 開始時刻 | 12:40（固定） |
| 終了時刻 | 13:20（固定） |

---

## 児童の並び順ルール

- 基本: **ランダム順**（フルネームで記載）
- 以下の3名は、迎え/送りの**最後尾**に配置する:
  - **寺田菫**
  - **園田柚真**
  - **園田葵**
  - 理由: 他施設を利用しているため、迎えに行く順序で最後になる

---

## 特殊条件

### 寺田菫・園田柚真・園田葵

- 他施設を利用しているため、迎えの順番は常に最後尾
- これらの児童がいる日は、夕迎えの終了時刻を **18:40固定** にする

### 櫻井麻智・櫻井斗麻（桜井あさと・カイト）

- デイサービスが夕方に「よりまど」まで送ってくれていた
- **月曜日・水曜日は夕方送迎（迎え）が不要**
- 2026年3月分までは在籍（PDFで在籍確認が必要）
- **2026年2月のデータには存在しない** → 退所済みの可能性あり、要確認

## 経路

- 起点・終点は固定値なし（朝と夕方に一回ずつ記録）
- 現行CSVでは経路列は省略（紙の帳票に手書きで記入する想定）

---

## 紙の帳票との対応

紙の送迎記録表には以下の構成がある（スプレッドシートに転記時の参考）:

### 左側（日別サマリ）

| 紙の列 | CSV対応 |
|---|---|
| 日付 | 日付 |
| 曜日 | 曜日 |
| 送迎職員 | 送迎職員（只津） |
| 介助者 | 介助者（溝口） |
| 送迎方法（徒歩・自動車） | 送迎方法（車） |
| 車番 | 車番（707） |
| 点呼確認による異常 | 点呼確認による異常（なし） |
| 利用者 | 児童名（順番順に列挙） |

### 右側（送迎詳細）

| 紙の列 | CSV対応 |
|---|---|
| 利用者（矢印つなぎ） | 児童名を順番順に「→」でつなぐ |
| 経路 起点・終点 | （手書き） |
| 送迎時間 開始 | 開始時刻 |
| 送迎時間 終了 | 終了時刻 |
| 備考 | 備考 |
