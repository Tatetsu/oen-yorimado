# gas-view — ダミーデータ ビューシート

## 概要

実績報告書・重度支援加算のCSVデータをスプレッドシートにインポートした後、
児童名・年月・書類タイプを選んでフィルタ表示するGAS。

## セットアップ

1. スプレッドシートに2つのシートを作成し、CSVをインポート:
   - `実績報告書` シート ← `実績報告書_全児童.csv`
   - `重度支援加算` シート ← `重度支援加算_全児童.csv`
2. スクリプトエディタ（拡張機能→Apps Script）に `view.gs` を貼り付け
3. メニュー「ダミーデータ」→「初期設定」を実行

## 使い方

ビューシートの3つのドロップダウンを変更すると自動更新:
- **B1**: 児童名
- **B2**: 対象年月（YYYY-MM）
- **B3**: 書類タイプ（実績報告書 / 重度支援加算）

手動更新: メニュー「ダミーデータ」→「表示更新」

## ビュー内容

### 実績報告書ビュー
| 列 | 内容 |
|---|---|
| 日付 | YYYY-MM-DD |
| 体温 | 36.0〜37.5 |
| 夕食・朝食・昼食 | 完食/半分/食べなかった/− |
| 入浴・便 | ○/× |
| 服薬(夜)・服薬(朝) | ○/× |
| その他連絡事項 | テキスト |

### 重度支援加算ビュー
| 列 | 内容 |
|---|---|
| 日付 | YYYY-MM-DD |
| 17:00〜8:00 | 毎時チェック ○/× |
| 12:30 | ○/−（他サービス併給時は−） |
| 睡眠チェック | 21:00(or 22:00), 4:00, 7:00 |
